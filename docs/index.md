# Botkit

## Installation

Get botkit from NPM

`
npm install --save botkit
` 


## Basic Usage

`
var Botkit = require('Botkit');

var bot = Botkit(configuration);
bot.init();

`

* RTM api / bot users (real user that receives all messages)
